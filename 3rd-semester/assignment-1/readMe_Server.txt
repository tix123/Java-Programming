Battleship Server 

This is a server program for a battleship game.
With this program, you can allow player connect 
to this server with battleship client program.

Date: 10/7/2020
Author: HSIAO YU CHEN (Sean Chen)

How to run the program:
1. For Windows 10, open "Command prompt"
2. Change to the folder where the server.jar is
3. Type "java -jar server.jar"
