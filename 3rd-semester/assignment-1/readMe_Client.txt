Battleship Server 

This is a client program for a battleship game.
With this program, you can connect to battleship 
server and play battleship game with other people.

Date: 10/7/2020
Author: HSIAO YU CHEN (Sean Chen)

How to run the program:
1. For Windows 10, open "Command prompt"
2. Change to the folder where the client.jar is
3. Type "java -jar client.jar"
