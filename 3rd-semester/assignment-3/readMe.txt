XML Parser

This is a XML Parser program. With this program, 
you can check XML file, and it will show error 
tags if it has any.

Date: 11/20/2020
Author: HSIAO YU CHEN (Sean Chen)


How to run the program:

1. For Windows 10, open "Command prompt".

2. Change to the folder where this Parser.jar is located.

3. Type "java -jar Parser.jar" with file name and path. 
   Example: "java -jar Parser.jar res/sample2.xml"
