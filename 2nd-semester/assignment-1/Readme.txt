Movie Management System

This program contains a movie list. With this program, you can
add a movie into the list, generate a movie list by year and
generate a random movie list.  

Date: 1/19/2020
Author: HSIAO YU CHEN (Sean Chen)


How to run the program:
1. For Windows 10, open "Command prompt"
2. Change to the folder where the H.Chen1.jar is
3. Type "java -jar H.Chen1.jar"

Notice: You must execute this file in its folder
	and that folder has a "res" folder which
	has a "movie.txt" file.
	
	i.e.:	\H.Chen1.jar
		\res\movie.txt
