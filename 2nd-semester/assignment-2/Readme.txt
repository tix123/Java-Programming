Book Management System

This is a book management system program. With 
this program, you can check out a book, find 
books by title, display books by type, and
generate a random book list.  

Date: 2/13/2020
Author: HSIAO YU CHEN (Sean Chen)

How to run the program:
1. For Windows 10, open "Command prompt"
2. Change to the folder where the H.Chen1.jar is
3. Type "java -jar H.Chen1.jar"
