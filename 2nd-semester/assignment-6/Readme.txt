Movie Management System

This program contains a movie list. With this program, you can
add or delete a movie into the list, generate a movie list by year 
and generate a random movie list.  

Date: 4/13/2020
Author: HSIAO YU CHEN (Sean Chen)

How to run the program:
1. For Windows 10, open "Command prompt"
2. Change to the folder where the H.Chen6.jar is
3. Type "java -jar H.Chen6.jar"
