Flight Reservation System

This is a flight reservation system program.
With this program, you can find flights, make 
a reservation, search reservations, and
update a reservation.  

Date: 3/22/2020
Author: HSIAO YU CHEN (Sean Chen)

How to run the program:
1. For Windows 10, open "Command prompt"
2. Change to the folder where the H.Chen4.jar is
3. Type "java -jar H.Chen4.jar"
